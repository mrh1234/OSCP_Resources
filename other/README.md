Located here is a bash script that you can use to scan for all open TCP ports with nmap quickly. For all open TCP ports found with the quick scan, the script will run an nmap service scan as well execute all standard scripts. Finally, the script scans nmap scan for the top 100 most common UDP ports. 


You can use the _set\_title.sh_ script to change the name of the terminal tab. This is helpful when you tend to use many terminal tabs at once.
